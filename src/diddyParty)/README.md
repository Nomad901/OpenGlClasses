# CMake_Setups
Useful CMake setups in order to just copy-paste  <br/>

to compile each:  <br/>
go to cmd. set path to the main folder, where is CMakeLists, include folder and so on. <br/>
Write in the cmd: <br/>

mkdir build <br/>
cd build <br/>
cmake .. <br/>
